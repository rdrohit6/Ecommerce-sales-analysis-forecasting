# E-Commerce Sales Analysis Forecasting

## 📌 Overview
This project performs an end-to-end analysis of an e-commerce dataset to uncover key business insights related to sales performance, customer behavior, and product trends. It also incorporates a forecasting model to predict future revenue patterns.

## ⚙️ Tools & Technologies
- Python
- Pandas, NumPy, Matplotlib
- SQL (SQLite)
- Microsoft Power BI
- Scikit-learn

## 📊 Project Workflow
- Data Understanding & Preparation
- Exploratory Data Analysis (EDA)
- SQL-based Analysis (Joins & Aggregations)
- Interactive Dashboard Development (Power BI)
- Sales Forecasting using Linear Regression

## 📈 Results
The Linear Regression model successfully captured the overall upward trend in revenue, achieving a Mean Absolute Error (MAE) of approximately 264K, establishing a baseline for future forecasting improvements.

## 🔍 Key Insights
- Revenue exhibits a consistent growth trend with noticeable seasonal variations
- A limited number of product categories contribute disproportionately to total revenue
- Sales are highly concentrated in top-performing states
- Customer-to-order ratio indicates relatively low repeat purchase behavior
- The observed decline in the final month is attributed to incomplete data rather than actual performance drop

## 🚀 Conclusion
This project demonstrates a comprehensive data analytics workflow, integrating data exploration, SQL-driven analysis, visualization, and predictive modeling. The insights derived can support data-driven decision-making in areas such as product strategy, regional targeting, and customer retention.
